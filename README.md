# Console logical mini-games brain-games

## Installation

### To install

1. clone repo
2. run
    - `$ make install`
    - `$ make build`
    - `$ make package-install`

### To uninstall

 run `$ make package-uninstall`


## Even number

The user must answer whether given number is even or not?


**Command to start the game:**  `$ brain-even`
